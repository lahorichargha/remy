Remy: A digital cooking assistant
Execution and setup instructions

BATCH FILE:
  1. Execute batch file to compile, assemble and run Remy:
        runRemy.bat
  
MAVEN:
 1. Assemble project with package command:
        mvn package

ECLIPSE: 
  1. Run "mvn eclipse:eclipse" to set up project files.
  2. Import into workspace
  3. Use VM arg -mx1000m and run Main "com.tastycode.remy.Remy"


After running remy:
 Say "What's next"
 Say "What do I do with the flour?"
 Say "Set a timer for 10 minutes"
Also feel free to type text to remy
